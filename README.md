# miZy ffmpeg 

miZy cross-compile example for custom FFMPEG arm

## Get Source Prepare and Build

    mkdir /tmp/zero_builder
    cd /tmp/zero_builder

    git clone https://github.com/hyphop/miZy-ffmpeg.git
    cd miZy-ffmpeg

    ./prepare
    ./BUILD_FFMPEG_A 


## x86 debian test

    ./prepare.x86
    ./BUILD_FFMPEG_X86
